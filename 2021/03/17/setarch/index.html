Linux Command –

setarch: change reported architecture in new program environment and set personality flags

setarch arch [options] [program [arguments…]]

options

–list: list all architecture setarch knows.
–uname-2.6: cause the program to see a kernel version since 2.6
–verbose: be verbose
–3gb: specifies a program to use a maximum of 3gb address space.
–32bit: limit the address to 32 bits to emulate hardware
–fdpic-funcptrs: Treat user-space function pointers to signal handlers as pointers to address descriptors.
–addr-compat-layout: provide legacy virtual address space layout.
–addr-no-randomize: disables randomization of virtual address space.
–sticky-timeouts: This makes select(2), pselect(2), and ppoll(2) system calls preserve the timeout value instead of modifying it to reflect the amount of time not slept when interrupted by a signal handler.
–read-implies-exec: If this is set then mmap(3) PROT_READ will also add the PROT_EXEC bit - as expected by legacy x86 binaries.
–version: display version information
–help: display help text

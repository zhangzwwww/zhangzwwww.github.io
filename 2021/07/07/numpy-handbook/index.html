Numpy

ndarray

ndarray 是 numpy 数组的类型, 有几个重要属性。

  • ndim 数组的维度的个数
  • shape 数组的维度，shape 元组的长度就是 ndim 的值。
  • size 总元素的个数，相当于 shape 的乘积。
  • dtype 数组中元素的类型

数组创建

调用 arrary() 方法用 python 的列表或元祖中的元素创建

使用 ones 或 zeros 将数据内容用 1 或 0 填充，使用 empty() 来随机数组内容

使用 arange([start ], stop, [step]) 能创建类似 range 方法的数组

打印数组

打印的时候 numpy 会自动跳过数组的中心部分，使用set_printoptions选项来禁用这个行为。

索引，切片和迭代

一维的数组能够进行索引，切片和迭代操作和python中的序列一样。

在多维数组中每个轴可以有一个索引，索引之间用逗号隔开。

当提供的索引缺少轴的时候，确实的索引被认为是完整的切片。

形状操作

以下的方法会改变数组的形状但是都是返回一个修改后的数组不会改变原始数组。

reshape方法不会修改原来的数组但是 ndarray.resize 方法会修改原来的数组。

在 reshape 中将size指定维-1，则会自动计算其他size的大小
